 This Project captures the Data layer of the three tiered architecture.

  The folders contain the following types of files:
 -------------------------------------------------

 * Interfaces	: contains interfaces internal to the Data layer classes
 * Entities		: classes that encapsulate actual entities that are involved
 * Mappings		: Classes that act as the repositories for each of the entities
 * Repositories	: base classes that provide repository logic/functions
